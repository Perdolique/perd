# Perd

<https://perd.perd.workers.dev/>

Perd is an outdoor equipment companion for people who plan trips, hikes, and other outdoor outings and do not want to forget important gear.

## What you can do now

- Sign in.
- Browse equipment.
- View item details.
- Save gear.
- Submit gear for review and use it in your inventory immediately.
- Create packing lists.
- Add checklist items.
- Mark items packed.
